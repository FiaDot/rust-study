## 학습 방법

### 매일 10분 루틴

```
1. 해당 모듈 파일 열기 (3분)
   - 코드와 주석 읽기
   - C++ 비교 부분 확인

2. cargo run으로 실행 (2분)
   - 출력 결과 확인

3. 코드 수정해보기 (5분)
   - 변수 바꿔보기
   - 에러 유발해보기
   - 컴파일러 메시지 읽기
```

### 실행 명령어

```bash
# 전체 실행
cargo run

# 특정 모듈만 실행하려면 main.rs에서 다른 모듈 주석 처리

# 테스트 실행
cargo test

# 특정 테스트만
cargo test test_name
```

### 복습 방법

1. **에러 유발**: 일부러 빌림 규칙 위반해보기
2. **수정 연습**: 함수 시그니처 바꿔보기
3. **작은 기능 추가**: 예제에 기능 하나 추가해보기

## 우선순위 (시간 부족시)

C++20 개발자가 **반드시** 이해해야 할 핵심 모듈:

1. `_02_ownership.rs` - 소유권 (Rust의 핵심)
2. `_03_borrowing.rs` - 빌림 규칙
3. `_06_enums.rs` - Option, Result, 패턴 매칭
4. `_09_error_handling.rs` - ? 연산자, 에러 처리
5. `_07_traits.rs` - 트레이트 (다형성)

이 5개만 확실히 이해하면 나머지는 C++ 지식으로 빠르게 습득 가능합니다.
