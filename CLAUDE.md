## Project Overview

This is a Rust learning/study project using the 2021 edition.

## Build Commands

- **Build**: `cargo build`
- **Run**: `cargo run`
- **Test**: `cargo test`
- **Run single test**: `cargo test test_name`
- **Check (fast compile check)**: `cargo check`
- **Format code**: `cargo fmt`
- **Lint**: `cargo clippy`
